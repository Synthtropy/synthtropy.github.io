Drum Tracks

Pattern-based drum tracks built from presets, with a complexity control for instant variation.

Drum tracks are pattern-based: instead of drawing every hit, you start from a template and shape it. A drum track can be powered by the built-in drum synthesizer, the drum sampler, or a hosted AUv3 plug-in.

Drum presets

Create a drum track from a preset to get a ready-made pattern and kit as a starting point.

Complexity

Each drum clip has a complexity slider that varies the pattern — thinning it out or filling it in — so you can dial in the busyness of a part without editing individual notes.

In this section

  • Drum Sampler — load your own samples into drum slots.
  • Drum Editor — edit the pattern directly.
